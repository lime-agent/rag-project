# 기술 스택 결정 기록

## 임베딩 모델: sentence-transformers

**결정**: `paraphrase-multilingual-MiniLM-L12-v2` 사용

**이유**:
- 로컬 실행 가능, 무료
- 한국어 포함 50개 언어 지원
- 모델 크기 약 400MB로 적당함
- OpenAI Embedding API 대비 비용 없음

**트레이드오프**: OpenAI text-embedding-3-small보다 품질 낮을 수 있으나 수업용 토이 프로젝트에 충분

## 벡터 DB: ChromaDB

**결정**: ChromaDB 로컬 파일 모드 사용

**이유**:
- 서버 없이 로컬 파일(`.chroma/`)에 저장
- `pip install chromadb` 한 줄로 설치
- Python API 단순함

**비교 검토**:
- FAISS: 설치 복잡, 우리 규모에서 성능 차이 없음
- Pinecone: 유료, 외부 서버 필요
- Weaviate: 설정 복잡

## LLM: Ollama (qwen2.5:3b)

**결정**: Ollama로 로컬 LLM 실행

**이유**:
- 외부 API 키 불필요
- 사내망에서도 동작
- qwen2.5:3b는 3B 파라미터로 속도·품질 균형

**트레이드오프**: Claude API 대비 답변 품질 낮음. 6회차에서 Claude API로 교체 예정
